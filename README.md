# Apollo

Milestone 2 includes completed frontend for home, problem, leaderboard, welcome, leaderboard and signup pages. Backend server and database user login authentication and verification has also been completed. Next steps will be fetching problem and user data from database.

Milestone 3 now has complete backend integration for fetching problems, user data, leaderboard data, testing and running problems. A new matchmaking page frontend was added as proof of concept and some UI changes were made overall.

Note:
To run the project, a .env file in the backend folder must be created which sets the DATABASE_URL variable to the correct connection string. Additionally for the AI assistant to work a GROQ_API_KEY must be set. Next the backend must be built and the server ran then on another terminal the GUI built and ran so the login features can work properly. For the problem testing to work, docker must be installed and configured on your system.
