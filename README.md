# Apollo

Milestone 1 includes completed frontend for home, problem, leaderboard, welcome, leaderboard and signup pages. Backend server and database user login authentication and verification has also been completed. Next steps will be fetching problem and user data from database.

Note:
To run the project, a .env file in the backend folder must be created which sets the DATABASE_URL variable to the correct connection string. Next the backend must be built and the server ran then on another terminal the GUI built and ran so the login features can work properly.
